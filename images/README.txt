PLACEHOLDER IMAGES - REPLACE THESE FILES
=========================================

The following image files need to be replaced with actual images:

1. images/headshot.jpg
   - Your professional headshot for the homepage hero section
   - Recommended size: 600x800px (3:4 aspect ratio)
   - Used on: index.html

2. favicon.png
   - Browser tab icon
   - Recommended size: 32x32px or 16x16px
   - Used on: all pages

3. apple-touch-icon.png
   - Icon for iOS devices when saved to home screen
   - Recommended size: 180x180px
   - Used on: all pages

4. og-image.png
   - Social media preview image (Facebook, LinkedIn, etc.)
   - Recommended size: 1200x630px
   - Used for: social sharing meta tags

To replace:
Upload your actual images with the same filenames to the repo root for
`favicon.png`, `apple-touch-icon.png`, and `og-image.png`, and to `/images`
for `images/headshot.jpg`.
